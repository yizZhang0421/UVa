純暴力，但如何暴力需要思考一下</br>
</br>
EX:題目範例</br>
A、B 兩種面額的郵票可以貼 3 張，AB為哪兩個面額可以得到最大連續總面額？</br>
像是 A=1，B=3</br>
1=1</br>
2=1+1</br>
3=3</br>
4=3+1</br>
5=3+1+1</br>
6=3+3</br>
7=3+3+1</br>
最大可以連續到7</br>
答案為"  1  3 ->  7"</br>
</br>
暴力遞迴關鍵在於嘗試的範圍</br>
暴力嘗試所有種面額</br>
第一個面額一定要1，不然無法形成任何連續</br>
下一個要嘗試的面額為：上一個最新嘗試的面額+1 ~ 目前所存在的所有面額所能產生的最大面額+1</br>
底限制可以理解，就是至少要大於目前的面額</br>
頂限制是因為+2會出現斷層，之前的最大將無法湊出來</br>
</br>
該例題模擬；</br>
面額：[1]，能產生的最大：3，下一個面額能嘗試[2, 3, 4]</br>
暴力搜到 3 可以有最大連續</br>
